# Intro

The ESP8266FZ project is inspired by the well known FlipperZero. This board uses a Wemos D1 mini 4.0.0 and is capable to play some low end games and IoT connectivity. The main purpose is to be highly adaptable to the users and act as a first step into micro controller programming.
Files for a chassi is also provided where the necessary components are a atleast four 14mm long M3 bolts together with 4 M3 nuts.
A demo code is also avaible for users to get a first insight into its capabilities.
